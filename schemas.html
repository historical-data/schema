Organization of Schemas

The schemas are a set of 'types', each associated with a set of properties. The types are arranged in a hierarchy, which inherits and shares more general types and properties with schemas listed at schema.org.

Historical and genealogical schemas:

  • Historical Document
  • Historical Event
  • Historical Family
  • Historical Person

Please see schema.org for more information about the general data types and properties.
