Organization of Schemas

The historical-data schemas are a set of enhancements to existing schema.org 'types' and new 'types', each associated with a set of properties. The types are arranged in a hierarchy, which inherits and shares more general types and properties with schemas listed at schema.org.

Historical and genealogical schemas:

  • HistoricalRecord
  • Event enhancements
  • Family
  • Person enhancements
  • SourceInfo

Please see schema.org for more information about the general data types and properties.
